# PRODIGY_BD_01

## Basic REST API with CRUD Operations

A REST API built with Django REST Framework that performs CRUD operations on users using in-memory storage.

---

## Features

- Create User
- Get All Users
- Get User By ID
- Update User
- Delete User
- UUID Support
- Input Validation
- Error Handling
- Swagger Documentation

---

## Technologies Used

- Python
- Django
- Django REST Framework
- drf-spectacular

---

## Installation

```bash
git clone <repository-url>
cd PRODIGY_BD_01

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python manage.py runserver
